Bonus!

1. Originally, our code put any uppercase letter before any lowercase letter (e.g. "Z" went before "a"). We assume this is because numeric values for uppercase ASCII characters are smaller than those for lowercase ones. But we fixed this! Thanks to ZunTzu on stackoverflow, we found a couple javascript tricks to make this possible. Instead of the following code to sort the text:

userNameSplit.sort();

We used this code instead:

userNameSplit.sort((a,b) => a.localeCompare(b, undefined, {sensitivity: 'base'}))

The sort method takes a function as an argument for how it should sort the array it's given. In this case, we gave it an arrow function that compares elements in a case-insensitive way.


2. Our code placed the spaces at the beginning of the array. We made our code omit spaces by using the filter array method:

nameFinalSpaces = userNameSplitSort.filter((value, index, arr) => value != ' ');

This method takes an existing array and a callback function and creates a new array based on what the callback function allows. Our callback function allows anything that isn't an empty space.
